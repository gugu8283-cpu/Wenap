# Wenap 转化率心理学优化清单

> 来源：虚拟用户审查 × 5 轮心理学迭代  
> 目标：免费 → 付费转化率 5–10% → 35–45%  
> 状态：见下方「实现进度」

## 心理触发器叠加原则

最高转化漏斗应叠加：**稀缺感 + 社会证明 + 损失厌恶**，而非单一杠杆。

| 原理 | 要点 |
|------|------|
| 损失厌恶 | 文案优先「你会损失什么」，而非「你能得到什么」 |
| 先给予价值 | 先真正帮助用户，再要求付费 |
| Aha 时刻 | 用户离开前触发；步骤越少完成率越高（3 步 >> 7 步） |
| 前 3 秒 | 落地页立刻传达价值/价格锚点，不堆功能列表 |

## 文案对照（损失框架）

| 场景 | 旧 | 新 |
|------|----|----|
| 涨幅 | 涨幅 +27% | 不看这份分析，可能错过约 $61 的上行空间 |
| 免费额度 | 还剩 2 次免费分析 | 还剩 2 次 · 用完后将失去对 NVDA 的实时跟踪 |
| Pro 升级条 | 升级 Pro | 没有 Pro，你在用不完整的信息做决策 |
| 批评家锁定 | 升级 Pro+ 查看 | 还有 N 条未读 · 可能正是你亏钱时忽略的风险 |
