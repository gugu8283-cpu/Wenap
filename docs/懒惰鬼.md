# 懒惰鬼（Wenap 代测约定）

> **懒惰鬼** = 你懒得自己一遍遍点分析、对语言、查报错；交给我用 **API 当机器人用户** 测，你只盯界面美不美。

---

## 分工

| 你（人眼） | 懒惰鬼（我） |
|------------|--------------|
| 界面好不好看、排版、动效 | 登录 free / pro / pro+ 测试号 |
| 按钮、菜单、截图里的 UI 文案 | 各 **资产类型**、**locale** 跑分析 |
| 可选：发 1～2 张截图补 UI 问题 | 读 **报告正文**：错语言、缺维、胡说、价格怪、SSE 报错 |
| | 输出：**bug 清单**（现象 + 可能原因） |

---

## 你怎么叫我开工

发一句即可，例如：

- `懒惰鬼 开始`
- `懒惰鬼 Pro+ NVDA en+zh-CN`
- `懒惰鬼 全资产抽样`

**范围写清楚越好**（档位、标的、语言）。不说就用默认：Pro+、`NVDA` + `BTC`、`zh-CN` + `en`。

---

## 你怎么让我停

- `懒惰鬼 停` / `停`

---

## 我不做的（别指望懒惰鬼）

- 评判 UI 美丑（除非带截图问「这按钮在哪」）
- 未经你同意跑几十次全量分析（费钱、费时）
- 代你收邮件验证码、Stripe 实付

---

```text
⚠️ analyze:EURUSD — data_warning: ...
```

## 上线前建议抽检（对应 PRODUCT-ROADMAP 对策表）

| 场景 | 查什么 |
|------|--------|
| `locale=en` + NVDA | 正文无大段中文 |
| `locale=ja` + 日股 | 币种/用语合理 |
| `locale=zh-CN` + BTC | crypto 维度不像硬套股票 |
| Pro+ + 美股 | 六维齐全、现价与 Hero 一致 |
| 错误路径 | 额度用尽、无效 ticker 返回可读错误 |

全功能上线后见 [PRODUCT-ROADMAP.md § 对策检查](./PRODUCT-ROADMAP.md#各阶段上线前必勾对策检查)。
