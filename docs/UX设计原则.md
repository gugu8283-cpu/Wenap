# Wenap UX 设计原则

> 主动优化依据：fintech 可读性、3 秒落地法则、暗色界面亮度、转化心理学（见 `转化率心理学优化.md`）

## 视觉层次

1. **背景**：不用纯黑 `#000`，基底色 `#12141c`，卡片 `#1a1e28`，减少眼疲劳  
2. **文字**：次要文案 `#a8b0bd`，辅助 `#7a8494`（禁止 `#444` 级不可读灰）  
3. **卡片**：1px 边框 + 轻阴影，模块边界清晰  
4. **数字**：`DM Mono` + tabular-nums；标题 `Syne`

## 交互

1. **主 CTA**：全宽绿色渐变按钮，唯一主行动  
2. **快捷标的**：NVDA / AAPL / GLD / SPY / TSLA 一键填入  
3. **焦点**：键盘 `focus-visible` 蓝色环  
4. **动效**：尊重 `prefers-reduced-motion`

## 报告页

1. Hero 区轻渐变底，分数环 glow 随涨跌色  
2. 雷达图网格使用 `--radar-grid`，亮色模式自动变浅  
3. 桌面端报告区 `max-width` 放宽、sticky 侧栏

## 落地页

1. 径向渐变背景 + 品牌绿点缀  
2. 准确率胶囊、样例链接居中  
3. 与 App 共用 CSS 变量，亮/暗一致
