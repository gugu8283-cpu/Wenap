# Wenap 日本税务 · Stripe 与税理士指南

> **免责声明**：本文仅供个人备忘与理解框架，不构成税务·法律意见。具体金额与义务请以国税厅最新规则及税理士判断为准。  
> **适用场景**：日本 Stripe 账户、网站订阅（wenap.app）、Stripe 选「右边」（自行管理税，非 Managed Payments +3.5%）。

---

## 1. 三种税，别混在一起

| 税种 | 是什么 | 日本 Stripe「右边」 | 日本 Stripe「左边」+3.5% |
|------|--------|---------------------|-------------------------|
| **所得税** | 赚了多少钱（扣除经费后）要交的税 | **自己**确定申告/法人申告 | **同样自己** |
| **日本消费税**（10%） | 对日本国内客户卖服务的间接税 | **自己**判断是否课税事业者、是否加 10% | **同样自己** |
| **海外 VAT/销售税** | 欧盟、英国、美国各州等对数字服务的税 | **自己**或 **Stripe Tax** / 税理士 | Stripe 在支持范围内**代征代缴** |

**Stripe 不会替你报日本所得税，也不会包办全部日本消费税实务。**

---

## 2. Stripe 左边 vs 右边（一句话）

| | 右边（推荐起步） | 左边（+约 3.5%/笔） |
|--|------------------|---------------------|
| **你少操心什么** | — | 海外 VAT/销售税（支持国） |
| **你仍要操心** | 所得税 + 日本消费税 + 海外税（或以后 Stripe Tax） | **所得税 + 日本消费税** |
| **有没有法定「营业额到 X 换边」** | **没有**；看 3.5% 成本 vs 税理士/OSS 成本 |

**粗算 3.5% 成本（月订阅收入）**

| 月收入（约） | 左边多付/月（约） |
|--------------|-------------------|
| 10 万日元 | ~3,500 日元 |
| 50 万日元 | ~17,500 日元 |
| 100 万日元 | ~35,000 日元 |

## 3. 日本所得税 — 门槛与处理方式

### 3.1 什么时候要「申报」（和有没有利润不完全等同）

| 你的身份 | 常见门槛 | 处理方式 |
|----------|----------|----------|
| **法人（株式会社等）** | 有法人 → **每年**法人税申告 | 税理士 / 法人申告 |
| **个人 + 副业**（另有日本工资） | 工资以外所得合计 **> 约 20 万日元/年** → 一般要**确定申告** | e-Tax・確定申告書 / 税理士 |
| **个人 + 主业（個人事業主）** | 营业中 → **基本每年**确定申告 | 青色/白色申告、税理士 |

### 3.2 什么时候「所得税金額」会明显变多

| 概念 | 大约金额 |
|------|----------|
| 应税所得 | **约 48 万日元以下** → 所得税 often 很少或 0（**住民税另算**） |
| 以上 | 累进税率（约 5%～45% 级距） |

### 3.3 日常处理流程

1. **Stripe** → 每月导出 CSV（日期、金额、手续费、客户国）。  
2. **台账**：收入表 + 经费表（Render、OpenRouter、域名、Stripe 手续费等，**保留发票**）。  
3. **每年 2～3 月**：确定申告（或法人申告）；美元按规则换算日元。

## 4. 日本消费税 — 门槛与处理方式

### 4.1 客户在哪里

| 客户 | 常见处理（需税理士确认） |
|------|--------------------------|
| **日本国内** B2C | 课税事业者 → 一般 **10%**；免税事业者 → 通常不征 10% 但要记账 |
| **日本国外** | 多按**对外提供役務** → 往往 **0%（日本消费税）**，要分客户记账 |

### 4.2 要不要登记「课税事业者」

| 条件（简化） | 大约金额 |
|--------------|----------|
| **基准期间**应税销售额 | **> 约 1,000 万日元/年** → 原则上须登记 |
| **特定期间**（1/1～6/30） | **> 约 1,000 万日元** → 可能当年 **10/1 起**按课税者 |

**免税事业者**：不做消费税申告，但须**监控销售额**是否逼近门槛。

### 4.3 インボイス（发票制度）

课税事业者后：对日本 B2B 等可能需要**适格发票**（登记番号 T+13 位）。

## 5. 海外 VAT — 门槛与处理方式（选右边时）

| 地区 | 大约门槛（B2C 数字服务） | 处理方式 |
|------|--------------------------|----------|
| **欧盟** | 跨境 B2C 常引 **约 1 万欧元/年** | OSS 登记 / **Stripe Tax** / 改左边 |
| **英国** | 常引 **£90,000/年**（规则以 HMRC 为准） | 税理士 / Stripe Tax |
| **美国** | **按州**，无全国统一数 | 体量大再评估 |

**起步**：欧盟远低于 1 万欧元 → **记账即可**；接近或超过再行动。

## 6. 总门槛一览表

| 事项 | 大约金额 | 怎么处理 |
|------|----------|----------|
| 副业确定申告 | 副业等 **> 约 20 万日元/年** | e-Tax / 税理士 |
| 主业个人确定申告 | 有营业 **基本每年** | 同上 |
| 法人税申告 | 有法人 **每年** | 税理士 |
| 所得税「要多交」 | 应税所得 **> 约 48 万日元** 起 | 申告时计算 |
| 消费税登记 | 销售额 **> 约 1,000 万日元/年** | 税務署届出 + 申告 |
| 欧盟 VAT（OSS） | **> 约 1 万欧元/年**（向欧盟 B2C） | OSS / Stripe Tax / 左边 |
| Stripe 换左边 | **无法定金额** | 比 3.5% 与税理士费 |

## 8. 推荐「最小流程」清单

**每笔订阅** → Stripe 入账 → 记入台账（日期 / USD / JPY / 手续费 / 国家 JP or 海外）

**每季度（约 15 分钟）** → 汇总日本 vs 海外销售额；整理经费发票

**每年 11～3 月** → 确定申告；是否消费税登记（税理士）；法人则法人税

**海外 VAT** → 第一年收入低只记账；欧盟变大 → Stripe Tax 或税理士

**网站** → https://wenap.app 、利用規約、プライバシーポリシー、非投资建议声明

**Stripe 本番** → `APP_PUBLIC_URL=https://wenap.app`、Webhook `https://wenap.app/api/billing/webhook`

## 9. 税理士 · 会計士 — 日语、差别、费用、能帮你什么

### 9.1 日语对照

| 中文 | 日语 | 英文 | 备注 |
|------|------|------|------|
| **税理士** | **税理士（ぜいりし）** | Tax accountant / Tax agent | 日本**税务**专业资格，可代理税务申告等 |
| **会计师（CPA）** | **公認会計士（こうにんかいけいし）** | Certified Public Accountant (JICPA) | 日本**注册会计师**，审计·会计·税务广 |
| **簿记** | **簿記（ぼき）** | Bookkeeping | 记账技能，**不等同**税理士资格 |
| **会计事务所** | **会計事務所（かいけいじむしょ）** | Accounting firm | 常同时有税理士·公認会計士 |

日常说「会计师」在日本指业务时，多数对应 **公認会計士** 或事务所里的 **税理士**（看具体担当资格）。

### 9.2 税理士 vs 公認会計士（差别）

| | **税理士** | **公認会計士** |
|--|------------|----------------|
| **核心** | 税法、申告、税务规划 | 财务报表、审计、企业会计全体 |
| **典型客户** | 个人事业、中小企业 | 上市公司、需审计法人、大企业 |
| **Wenap 起步阶段** | **通常先找税理士就够** | 收入极大、要审计、IPO 时才刚需 |
| **能否代理报税** | 可（在代理范围内） | 税理士登记范围内也可涉及税务 |
| **法律资格** | 税理士法 | 公認会計士法 |

**找谁**：Wenap 这种 **个人/小法人 SaaS + Stripe 订阅** → **税理士** 优先；不必一开始就请四大 CPA。

### 9.3 咨询一次大概多少钱（日本·2024～2025 量级）

**因都道府县·事务所·线上/线下差异很大，以下为常见区间：**

| 项目 | 大约费用（日元） |
|------|------------------|
| **初次相谈（30 分～1 時間）** | **0（免费）～ 11,000**；有的收 **5,000～15,000/时** |
| **确定申告委托（个人·白色·收入简单）** | **5 万～15 万/年** |
| **确定申告（个人·青色·副业+Stripe）** | **8 万～25 万/年** |
| **法人申告（小法人）** | **15 万～40 万+/年** |
| **月次顾问（记账+咨询）** | **1 万～3 万/月**（小事业） |
| **消费税申告追加** |  Often **+2 万～10 万/年** |

**线上税理士·免费初回** 较多；都内面对面偏贵。

### 9.4 税理士能帮你做什么（和 Wenap 相关）

| 服务 | 内容 |
|------|------|
| **初次咨询** | 个人 or 法人、副业 20 万规则、要不要消费税登记 |
| **记账指导** | Stripe 导出怎么记入、经费什么能扣 |
| **确定申告/法人申告** | 制作申告书、e-Tax 提交、计算税额 |
| **消费税** | 是否届出课税事业者、中间申告、インボイス |
| **税务规划** | 青色申告、经费合理化（合法范围内） |
| **海外 VAT 方向** | 何时考虑 OSS、Stripe Tax vs 左边（与跨境数字服务相关） |
| **被税務署询问** | 对应书信（额外费用） |

**通常不做**：帮你写代码、Render 部署、Stripe 技术对接（那是工程师的事）。

### 9.5 第一次去咨询带什么

1. 日本 Stripe **売上** 截图或 CSV（若有）  
2. **经费** 列表（Render、OpenRouter、域名等）  
3. 客户 **日本 / 海外** 比例（估计即可）  
4. 是否 **有工资**、是否 **法人**  
5. 问题清单（见下）

**可问税理士的三句话（日文例）**  
- 個人事業主として Wenap の Stripe 収入をどう申告しますか。  
- 消費税の免税事業者のままでよいか、登録が必要か。  
- 海外（EU）向けサブスクの VAT はどう対応すべきか。

### 9.6 哪里找

- **税理士検索**（日本税理士会連合会）  
- **freee・マネーフォワード** 提携税理士  
- 华人税理士（中文咨询，东京/大阪较多）  
- 英文 **English-speaking tax accountant in Japan**

## 10. 三条硬规则（备忘）

1. **副业 + Wenap 有收入** → 先假定超过 **20 万日元/年** 要准备**确定申告**。  
2. **销售额**（非利润）盯 **1,000 万日元/年** → **消费税**登记。  
3. **欧盟 B2C 订阅** 盯 **约 1 万欧元/年** → 海外 VAT；此前 **右边 + 记账** 通常够。
